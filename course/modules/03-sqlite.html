SQLite 与数据形状

表里的每一行像一张收据存根：固定栏目（金额、时间、类型），TypeScript 的 interface 就是存根的模板复印件。

App 如何把数据库文件带进整棵树？

Context 由 SQLiteProvider 创建；子组件里 useSQLiteContext() 能拿到同一个 db 句柄。

CODE 
          <SQLiteProvider
            databaseName="mySQLiteDB.db"
            useSuspense
            assetSource={{
              assetId: require("./assets/mySQLiteDB.db"),
            }}
          >

用官方封装包一层：下面所有屏幕共享这一个 SQLite。

磁盘上的文件名，第一次会从内置资源复制出来。

打开库未完成时交给 React Suspense 显示「正在加载数据库…」。

assetSource 告诉运行时：预置的 .db 文件在哪个 bundle 路径。

require 指向打包进 App 的二进制种子数据库。

对象字面量结束。
